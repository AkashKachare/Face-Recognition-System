# Face Recognition System

A Java-based Face Recognition System developed using Java, OpenCV, and JavaCV.

## Features
- Face Detection using Haar Cascade Classifier
- Face Capture using Webcam
- Dataset Training using LBPH Algorithm
- Real-time Face Recognition
- User Registration with Face ID

## Technologies Used
- Java
- Maven
- OpenCV
- JavaCV
- Swing GUI

## Project Workflow
1. Capture Face Images
2. Train Dataset
3. Generate LBPH Model
4. Recognize Faces in Real Time
